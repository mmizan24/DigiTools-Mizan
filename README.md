# DigiTools - Premium Digital Marketplace

DigiTools is a modern, high-performance digital tools marketplace designed for creators, entrepreneurs, and digital nomads. It features a curated collection of premium tools, a seamless shopping experience, and a sleek, responsive user interface.

## 🚀 Features

- **Dynamic Product Grid**: Browse a wide range of digital tools with detailed descriptions, pricing, and feature lists.
- **Interactive Cart System**: Add items to your cart with real-time updates and visual feedback ("Added to cart" button state).
- **Smooth Toggling**: Easily switch between the Product view and the Cart view.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices using Tailwind CSS and DaisyUI.
- **Real-time Notifications**: Integrated `react-toastify` for instant feedback on user actions like adding/removing items or checking out.
- **Modern Animations**: Powered by `motion` for smooth transitions and engaging UI elements.
- **Premium Aesthetic**: Custom-themed UI with a focus on typography (Plus Jakarta Sans) and a sophisticated color palette.

## 🛠️ Technologies Used

- **React.js**: Frontend library for building user interfaces.
- **TypeScript**: For type-safe code and better developer experience.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **DaisyUI**: Tailwind CSS component library for beautiful, accessible UI elements.
- **Motion**: Animation library for React.
- **Lucide React**: Beautifully simple, pixel-perfect icons.
- **React-Toastify**: For elegant toast notifications.
- **Vite**: Next-generation frontend tooling for fast development and builds.
